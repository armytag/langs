    - 0 → kuhpum
    - 1 → lhasthu
    - 2 → alhu
    - 3 → pamka
    - 4 → sthuum
    - 5 → psku
    - 6 → lhupthum
    - 7 → pthus
    - 8 → kau
    - 9 → kpthaku
    - 10 → kthskpthsm
    - 11 → kukptha
    - 12 → kthukpthah
    - 13 → lhutam
    - 14 → psuh
    - 15 → sthau
    - 16 → sthamtsh
- Each level of magnitude combines with a circumfix
    - 17^0 → s-...-sths
    - 17^1 → ka-...-kthah
    - 17^2 → ku-...-kths
    - 17^3 → pthah-...-pam
    - 17^4 → ta-...-kptha
- Some examples:
    - 666 (base 10) → 2 * 17^2 + 5 * 17 + 3 → kualhukths kapskukthah spamkasths
    - 666 (base 17) → kulhupthumkths kalhupthumkthah slhupthumsths → 1,842 (base 10)

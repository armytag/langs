# Junglelang

## Ideas

Reference Languages:
- Newar (Sino-Tibetan)
- Javanese (Austronesian)
- Sinhala (Indo-European)

Features:
- Breathy voice
- Retroflex & Palatal
- Reduplication
- Egophoric verb agreement
- Mixed morphosyntactic alignment

Day 87: March 27, 2024

Goal: Create plural indefinite forms

Note: Consider how your conlang may distinguish all versus some or many .

Tip: You can create more later on—these are just to get started.

Work focus: Create/Make/List

Today’s goal is to create more indefinite pronoun (and modifier) forms that indicate more one of a given entity, such as some, few, many, and all . As you create forms, it’s also good to consider how your language might work with these forms as modifiers, especially considering how all might treat the noun it occurs with differently than than some or many . It may require a different number inflection (or even case inflection) for the noun it occurs with, especially if you have inflections like a paucal or a collective in your language.

There are a lot of these forms you could create, so you don’t need to try to get it all done in a day. Rather, focus on getting a solid start so that later, if you want to add more, you already have a system in place to create and structure the forms.
